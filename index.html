Sobre Mim

Sou um desenvolvedor Full-Stack apaixonado por tecnologia, com mais de 3 anos de experiência em desenvolvimento de software. Tenho um grande interesse por resolver problemas complexos e estou sempre em busca de novas oportunidades para aprender e crescer. Possuo sólida formação em Engenharia de Software e habilidades em tecnologias modernas como AWS, Docker e bancos de dados, além de frameworks como Django e Django Rest Framework. Adoro colaborar em equipes ágeis e acredito no poder da inovação para transformar ideias em soluções práticas e eficientes.

Projetos

Análise de sentimentos

Overview: Django, SQLite, Docker & Python.


Um projeto Django de análise de sentimentos que analisa e classifica o sentimento expresso em frases submetidas, exibindo os resultados numa interface estilizada e responsiva. A aplicação foi desenvolvida com foco em usabilidade, proporcionando uma experiência de usuário intuitiva. Além disso, o projeto é facilmente extensível, permitindo futuras melhorias e integrações.

Backup Api

Overview: Python, API & BD.


Este repositório contém uma solução Python para o backup automatizado de dados de várias APIs diretamente para um banco de dados, visando a preservação e análise segura dos dados. Projetado para ser flexível, suporta diversos formatos de dados e sistemas de banco de dados, facilitando a integração e o gerenciamento de dados.

Sensor Ultrassônico

Overview: Sensor: HC-SR04, Arduino Uno & C++


Repositório para a integração do sensor Ultrassônico HC-SR04 com Arduino UNO. Inclui código-fonte, esquemas de conexão e guias para facilitar a configuração e uso do sensor em projetos de medição de distância e detecção de objetos. Ideal para entusiastas de eletrônica e estudantes.

ScreenMatch

Overview: Java - Encapsulamento, Polimorfismo e princípios básicos.


'Screenmatch' é um projeto pessoal desenvolvido para explorar e aplicar os princípios de programação orientada a objetos em Java. O foco do projeto é criar soluções eficientes para problemas específicos, utilizando práticas avançadas de desenvolvimento de software.
